About PlanetSVG

PlanetSVG connects vector creation and SVG animation in one private browser product. Start with the visual Editor, or open an existing file directly in the Animator.

Editor

Build artwork from parametric shapes with grid and object snapping, alignment guides, pathfinder operations, custom path editing, layers, undo and redo. Export SVG, PNG and multi-size ICO files.

Animator

Select individual SVG elements and add draw, fade, slide, scale, rotate or pulse tracks with duration, delay, easing and repeats. Scrub the timeline and export a self-contained animated SVG or its CSS.

Connected workflow and autosave

The Editor's “Animate this SVG” button transfers the current design locally into the Animator. Editable compositions and animation drafts are autosaved separately in browser storage, so refreshing or returning later restores the relevant workspace.

Privacy

Creation, handoff, animation and export happen locally. Artwork is never uploaded.
